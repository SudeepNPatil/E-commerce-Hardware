# E-Commerce Hardware
#### E-commerce Platform for Both Custom Build Pc's and Readymade Products with technicion Booking Support

## Description 
the plaform is designed to perform both Custom Build Pc's and Readymade Products and there is technician booking is there ,
this is where we stand out as new feature.

## features
- Authantication & Authorization
- Role base access
- CRUD operation
- Context for global state
- Cloudinary for image upload
- Admin Panal
- Add Products

## Tech Stack 
- React js
- TailwindCss
- MongoDB
- Express js
- Node js

## Installation 
```bash 
git clone (https://github.com/SudeepNPatil/E-commerce-Hardware.git)
cd E-commerce-Hardware
cd Frontend
npm install
npm run dev
for backend in new terminal
cd backend
npm install 
npm start

Create a .env file in the Backend and add:
DB_URL = your-Db_url_here
JWT= your_JWT_Screate_here
PORT = Port_you_want_to_Run

Create a .env file in the Frontend and add :
VITE_API_URL: your_backend_url

```
